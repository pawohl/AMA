    Filter and search application for systematic Thin-Layer-Chromatography

    Filter-und Suchanwendung für die Dünnschichtchromatographie

    Project maintained by pawohl Hosted on GitHub Pages — Theme by mattgraham

    Features

    Datenbankverwaltung

    Mit Qt und Standard-SQL-Syntax ist die Anwendung zukunftssicher auf nahezu allen Plattformen und mit allen Datenbank-Backends lauffähig.
    Alle zur Verwaltung der Stoffe benötigten Funktionen sind bereits in die Anwendung integriert. So können Datensätze importiert, exportiert, Strukturformeln aus Dateien geladen werden, Datensätze gelöscht oder neu hinzugefügt werden. Ein integrierter Strukturformeleditor ermöglicht das Hinzufügen neuer Datensätze im Handumdrehen.

    Übersichtliche Menüs und ein auf die Anforderungen zugeschnittener Funktionsumfang zeichnen diese Anwendung aus.

    Soft-Hinting: Ungültige Eingaben werden erkannt und hervorgehoben, Strukturformeleditor und Rich-Text-Editor erleichtern das Hinzufügen neuer Datensätze.

    Ansichten

    Ob Tablet, Smartphone, Laptop, Netbooks, Drucker oder der 24″ Monitor; Analysenbearbeitung oder Vorbereitung – Für alle gibt es die richtige Ansicht.

    Suche

    Ein übersichtlicher Suchdialog mit intelligenter Options-Auswahl und nützlichen Extras macht die Suche anhand vorliegender Analysenergebnisse zum Kinderspiel. Die Ergebnissortierung erfolgt durch einen eigens entwickelten Algorithmus nach Relevanz.

    Außerdem können Namen vergeben werden, so dass die Anwendung auch durch mehrere Benutzer genutzt werden kann. Die Ergebnisse werden in jeweils eigenen Tabs dargestellt.
